A simple, lightweight distribution for MIPS

Arch Linux MIPS is a Linux distribution built from the source of Arch Linux with minimal changes necessary to successfully build for & run on the MIPS architecture.

Currently the distribution is available for machines capable of executing mips32r2el hard float binaries (that is, little endian systems whose CPUs implement revision 2 of the MIPS32 ISA). Big endian builds are planned for the future.
